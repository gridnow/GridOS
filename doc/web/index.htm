GridOS介绍

GridOS操作系统是一款核心自主的面向未来网络应用的微内核操作系统，它将互联网视为软件的开发和运行平台，把软件无缝地从客户端设备延伸到巨大的互联网后台集群，把操作系统看作是一个实现高层次的资源管理与共享、支撑多样性和灵活性服务的透明并发运算平台，支撑多种新型网络应用模式，形成新型、开放、可持续生长的信息系统。

我们在操作系统开发上的尝试有很多成功，也有许多的失败。经过近10年的技术沉淀，GridOS将把所有的经验和技术精髓集于一身！GridOS正刚刚起步，并开放源代码。相信在大家共同的努力下，GridOS将真正成为下一代通用网格计算的操作系统，并在此基础上衍生出更高级别的软件运行和应用形式，最终把软件分子的新型软件模型呈现给用户，释放不同架构处理器和外围硬件组成的集群计算能力，为用户打造更加方便、高效、强大的计算基础软件平台。
